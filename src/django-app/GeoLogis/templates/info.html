Sourcing & Modèle

Notre modèle de classification est entraîné sur des jeux de données certifiés afin de garantir une précision optimale dans l'analyse immobilière.

  • • Data.gouv (DVF) : Utilisation de la base "Demande de Valeur Foncière" pour l'historique des mutations.
  • • INSEE : Intégration des données démographiques et socio-économiques par quartier.

Algorithme : Classification basée sur le Random Forest pour une segmentation précise des types de marchés locaux.
